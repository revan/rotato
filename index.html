Static Dashboard Rotation

Rotato is a no-code, low-config way to rotate webpages in a kiosk, written to cycle multiple DataDog dashboards on one balena-dash.

Configure using url parameters: delay to set the time in seconds between frames, and any number of url as optionally url-encoded pages to show. The pages are loaded in iframes and toggled in and out, avoiding reloads. Note that many sites will refuse to load in an iframe.

Example, rapidly switching between two urls: 

https://revan.io/rotato/?delay=3&url=https%3A%2F%2Ftwitframe.com%2Fshow%3Furl%3Dhttps%253A%252F%252Ftwitter.com%252Fjack%252Fstatus%252F20&url=https%3A%2F%2Ftwitframe.com%2Fshow%3Furl%3Dhttps%253A%252F%252Ftwitter.com%252Fdril%252Fstatus%252F922321981
